## Problem

The AI and crypto spaces are converging, but there's no consumer-facing product that lets users **experience** AI competition in a fun, tangible way:

1. **AI is a black box** — Users hear about GPT, Claude, Gemini, but can't see them compete head-to-head on real-world tasks.
2. **Prediction markets are boring** — Binary yes/no outcomes with no entertainment value. No real-time action.
3. **No AI × DeFi consumer apps** — Plenty of AI trading bots, but zero gamified spectator experiences with on-chain betting.
4. **Model evaluation is academic** — MMLU benchmarks don't translate to real users. There's no accessible leaderboard for "which AI is actually smarter at X?"

Users want to watch AI compete, bet on outcomes, and see real results — like esports, but for artificial intelligence.

---

## Solution

**GemBots Arena** is a live AI battle platform where frontier LLM models compete in real-time crypto price prediction battles. Users watch, bet BNB, and collect winnings.

### Key Innovations
- **Real LLM inference** — Every prediction is generated by an actual frontier model (Claude, GPT, Gemini, etc.) via OpenRouter. No simulations.
- **Spectator betting** — Watch AI fights live and bet BNB on the outcome via trustless smart contracts on BSC.
- **Model Leaderboard** — Transparent, data-driven ranking of which AI models are best at crypto forecasting.
- **Perpetual tournaments** — 24/7 non-stop battles. Always something to watch, always something to bet on.

## How It Works

### Battle Flow

```
┌─────────────────────────────────────────────────────────┐
│                    BATTLE LIFECYCLE                       │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  1. CREATE BATTLE                                        │
│     └─ Player deposits entry fee (BNB) → Smart contract  │
│                                                          │
│  2. JOIN BATTLE                                          │
│     └─ Challenger matches entry fee → Battle goes Active │
│                                                          │
│  3. AI PREDICTION ROUND (3 minutes)                      │
│     ├─ Both players' AI models receive live market data  │
│     ├─ Each LLM makes real-time price predictions        │
│     └─ Predictions are scored against actual movements   │
│                                                          │
│  4. RESOLUTION                                           │
│     ├─ Oracle submits winner to smart contract            │
│     ├─ Winner receives pot minus 2.5% platform fee       │
│     └─ All data recorded for Model Leaderboard           │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

### Tournament Format
- **8 bots** per tournament, single-elimination brackets
- Quarter-finals → Semi-finals → Finals
- 3-minute matches on real tokens
- Live streaming with chat and betting interface
- New tournament starts immediately after the previous one ends

## Revenue Model

GemBots generates revenue from **platform fees on every battle**, not speculation or token sales.

### Revenue Streams

| Source | Fee | Status |
|--------|-----|--------|
| 1v1 Battle fees | 2.5% of pot | ✅ Live |
| Tournament pool fees | 2.5% of pool | ✅ Live |
| Premium subscriptions | $9.99/mo | 📋 Planned |
| API access (high-volume) | Usage-based | 📋 Planned |

### Unit Economics
```
Per Battle (0.5 BNB entry each = 1.0 BNB pot):
  Revenue:  0.025 BNB (2.5% fee)
  Cost:     ~0.002 BNB (2x LLM inference via OpenRouter + gas)
  Margin:   ~92%

At 100 battles/day × 1.0 BNB avg pot:
  Daily revenue:  2.5 BNB (~$1,500)
  Monthly revenue: 75 BNB (~$45,000)
```

Revenue scales linearly with user adoption and betting volume.
